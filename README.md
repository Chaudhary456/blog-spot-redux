# Blog-Spot is React based web-application to manage Blogs.  
## Tech Used:  
###     Javascript  
###     React  
###     Redux  
###     JSON  
###     json-server  
###     HTML & CSS  
  
## JSON is used as a database to store blogs.
## We can perform Create, Read, Delete and Update operations on blogs.
